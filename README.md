## Installation ğŸ› ï¸

1. Clone the repository
```bash
git clone https://github.com/waseemhnyc/instagraph-nextjs
```
2. Navigate to the project directory
```bash
cd instagraph-nextjs
```
3. Install packages
```bash
yarn
```
4. Set environment variables
```bash
OPENAI_API_KEY=your-api-key-here
```
5. Run Next JS app
```bash
yarn run dev
```
